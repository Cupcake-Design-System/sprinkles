## Basic Loader

### Properties
| Name              | Type                      | Description                    | Default value         |
| ----------------- | ------------------------- | ------------------------------ | --------------------- |
| diameter          | rem value                 | diameter of the loader         | 16 rem                |
| thickness         | rem value                 | line thickness of the loader   | 0.3 rem               |


#### Usage
```html
<spr-loader-basic 
  diameter="5rem" 
  thickness=".25rem">
</spr-loader-basic>
```
